💡 How to Deploy

  1. Download this entire folder.
  2. Copy the two HTML files from 03-TOOLS-READY-TO-DEPLOY to a USB drive or email.
  3. Transfer to every store tablet.
  4. Double-click each file to verify it opens.
  5. Train store managers using the 10-Minute Quick Start.
  6. Print the Medical Patient Playbook for every register.

✍️ How to Edit

  1. Make a backup copy of any file before editing.
  2. Open the editing guide for the tool you want to change.
  3. Follow every step in order. Do not skip steps.
  4. Use Notepad (Windows) or TextEdit in plain text mode (Mac).
  5. Do not use Microsoft Word.
  6. Save. Test in a browser. Deploy the updated file.
